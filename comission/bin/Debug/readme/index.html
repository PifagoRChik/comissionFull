Для чайников
Если у вас остались вопросы после прочтения документации, вернитесь в программу и опишите свою проблему.

Описание программы

Данная программа предназначена для ведения учета абитуриентов в среднее специальное образовательное учреждение. Присутствует возможность добавления новых абитуриентов, прикрепления фотографий, просмотр общей информации о всех и детальной информации об отдельном абитуриенте.

Форма авторизации

Форма авторизации

На данной форме присутсвуют два поля ввода - логин и пароль. В эти поля вносятся соответствующие значения на английской раскладке клавиатуры. При отсутствии или неправильности введенных данных программа выдаст уведомление о ссответсвующей ошибке.

Рабочая форма

Рабочая форма

Данная форма является основной. На ней происходит основная часть работы программы. Изначально открывается элемент управления с добавлением нового абитуриента. Для добавления нового абитуриента необходимо внести информацию во все поля, прикрепить все фотографии и нажать на кнопку "Добавить абитуриента" При внесении неверных данных или при отсутствии таковых, программа выдаст уведомления о соответствующих ошибках.

Боковое меню

На боковом меню присутсвует выбор между вкладками, существующими в программе. Первая вкладка открывается со старта программы - "Добавление абитуриентов". На ней происходит добавление нового абитуриента в базу данных. Следующая вкладка - "Абитуриенты". На ней происходит просмотр информации о всех абитуриентах и ее редактирование. Последняя вкладка - "Помощь". Она открывает данное руководство и форму с отправкой вопроса.

Основые данные

На вкладке "Данные" есть возможность ввести номер паспорта абитуриента и на открывшейся форме ввести личные данные об абитуриенте.

Аттестат

На вкладке "Аттестат" доступно изменение данных аттестата абитуриента.

Абитуриенты

На вкладке абитуриенты при двойном клике по паспорту студента происходит открытие личной карточки студента, в которой можно изменять информацию о студенте. Над таблицей с ифнормацией о всех студентах присутсвует поле для поиска студента по фамилии. Для сброса результата поиска достаточно кликнуть в любом свободном месте программы.

Помощь

На вкладке помощи есть возможность связаться с разработчиками и описать свою проблему или предложения по работе программы.

По всем оставшимся вопросам сообщайте разработчикам во вкладке "Помощь"
